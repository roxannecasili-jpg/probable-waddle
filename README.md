# Roxanne Casili - Portfolio Website

A modern, continuous-scrolling portfolio website for Roxanne Casili, Social Media Manager & Virtual Assistant.

## 🎨 Design Features

- **Modern Design**: Professional dark theme with cyan/blue accents
- **Continuous Scrolling**: Smooth scroll experience with animated sections
- **Responsive Layout**: Fully responsive design for all devices
- **Interactive Elements**: Hover effects, animations, and smooth transitions
- **Professional Branding**: Aligned with VA/Social Media Manager aesthetic

## 📋 Sections

1. **Navigation Bar** - Fixed header with smooth scrolling links
2. **Hero Section** - Eye-catching introduction with CTA buttons
3. **About Me** - Personal background and statistics
4. **Services** - Six core service offerings with icons
5. **Portfolio** - Recent project showcase with descriptions
6. **Skills** - Comprehensive skills organized by category
7. **Why Work With Me** - Key differentiators
8. **Contact** - Multiple contact options and call-to-action
9. **Footer** - Footer with tagline

## 🚀 Technologies Used

- **HTML5** - Semantic markup
- **CSS3** - Modern styling with custom properties and animations
- **Vanilla JavaScript** - Interactive features and scroll effects
- **No Dependencies** - Pure frontend implementation
